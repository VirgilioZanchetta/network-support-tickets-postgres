\# Network Support Tickets – PostgreSQL Project



This repository contains a practical project focused on \*\*PostgreSQL database

administration and SQL analysis\*\* using a fictional \*\*network support ticket\*\*

management scenario.



The project is designed for \*\*technical learning, hands-on practice, and

professional portfolio demonstration\*\*.



\---



\## 🎯 Project Objectives



\- Practice PostgreSQL database creation and management

\- Apply SQL for operational and support analysis

\- Model a simple ticket/incident database

\- Demonstrate organization, documentation, and best practices



\---



\## 📂 Project Structure



\- `sql/` – Database schema, tables, sample data, and analysis queries

\- `docs/` – Project documentation and data dictionary



\---



\## 🧱 Database Overview



The database represents a \*\*fictional network support environment\*\*, including:



\- Ticket identification

\- Site and activity information

\- Timestamps and duration

\- Effort metrics

\- Ticket status



All data used in this repository is \*\*synthetic and anonymized\*\*.
